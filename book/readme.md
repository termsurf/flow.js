# Flow Book

This short set of notes ("book" used loosely) are compiled
thoughts/theory/implementation details on how to possibly implement
fuzzy and prefix text searching of words in a dictionary from across
multiple different languages (Sanskrit, Tibetan, Mandarin, English,
Hebrew, Arabic, Vietnamese, Akkadian, Pali, Latin, Greek, Turkish,
etc..). Each of these languages have their own quirks, so they must be
each considered separately for the most part, it turns out.
