## Hausa

A language game in Hausa involves inserting -bV- after the vowel of each
word-internal syllable. The vowel of the infix is a copy of the
preceding vowel.

## Yarok

```
-eg-

chyuuk'wen = to sit
chyeguuk'wen = to sit often

hookwch = to gamble
hegookwch = to gamble regularly

kemol = to steal
kegemol = to be a thief

laay = to pass
legaay = to pass regularly

chrwrhs = point
chuuwrh(s-) = point repeatedly

swrrk'ws = the wind gusts
swrgrrk'ws = the wind gusts repeatedly
```
